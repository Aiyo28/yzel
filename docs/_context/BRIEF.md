# Yzel (Узел) — Brief

## What

Yzel = MIT-licensed MCP connector portfolio (1C, Bitrix24, AmoCRM, Moysklad, Telegram, WhatsApp, Email/IMAP) bridging CIS business tools to AI. Public OSS layer for developer pull + integrator credibility. AIYO OS (closed cloud SaaS) sits on top as the revenue surface — cross-system entity matcher, context graph, and audit engine run **server-only** (trade-secret moat, not license moat). Target buyer: RU-speaking mid-upper businesses ($5–50M revenue) globally, with KZ O&G lighthouse for testimonial credibility.

## Current Phase

Phase 1 pivot (Apr 2026). Product+GTM restructured after grill session. Shipping Telegram/WhatsApp/Email connectors. Building AIYO OS SaaS landing. Zhenis-introduced Atyrau O&G lighthouse pilot as first paid account + case-study factory.
